# Sample Documents

This folder is for storing sample test documents for the Document Intelligence API.

## Supported Document Types

### Invoices
- PDF or image files containing vendor invoices
- Should have: invoice number, vendor name, line items, totals

### Bank Statements
- PDF or image files containing bank account statements
- Should have: account holder name, transactions, balances

### KYC Documents
- PDF or image files of identity documents
- Supported: Aadhaar, PAN, Passport, Driving License

## Adding Test Documents

1. Place your test documents in this folder
2. Name them descriptively (e.g., `sample_invoice.pdf`, `bank_statement_jan.png`)
3. Keep file sizes under 10MB

## Privacy Notice

Do NOT commit real documents with sensitive information to version control.
Use sample or anonymized documents for testing purposes only.
